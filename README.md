# 🌤️ AI Weather Stylist Bot

Telegram-бот, который анализирует погоду и даёт персональные советы по одежде с помощью ИИ.

## ✨ Возможности

- 🌡️ Анализ текущей погоды и прогноза
-  Персональные советы по стилю

## 🛠️ Технологии
Python 3.12 + Aiogram 3
OpenRouter (бесплатные LLM)
Open-Meteo API (погода)
FastAPI (для Mini App)
SQLite + SQLAlchemy
